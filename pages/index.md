# Welcome!
This is the default page for the Wrongo webring backend. If you see this page in a web browser, everything's probably set up correctly!

## What do I do now?
Start by customizing `/pages/config/preferences.json` to give your webring a name, then add your first members to `/pages/config/members.json`. Finally, edit this page source at `/index.md`, you can rename this file to end with `html` if you don't want to use markdown.

## I want more pages
Just add a new file (`md`, `html`) to the `/pages` directory, you may also want to append a link to the new page to the links section of `/pages/config/preferences.json`.
